Project Overview

Designed and authored a program that takes a CVS file containing raw NYC bike traffic data and extracts conclusions regarding weekly traffic patterns, weather dependencies, and finding the best implementation of traffic sensor installation.

Key Visuals

Estimated correlation values for weather events across NYC bike traffic. Full tables and analysis are included in the report below.

Technical Stack

  • Python
  • Pandas (CSV ingestion, cleaning, aggregation)
  • scikit-learn (LinearRegression, R² evaluation)
  • Matplotlib (bar chart visualization)

Engineering Approach

  • Imported and cleaned the NYC bicycle counts dataset.
  • Built and compared four 3-bridge linear regression models to determine the most effective sensor deployment strategy using R² as the selection metric.
  • Modeled traffic dependence on weather using multivariable regression (high temp, low temp, precipitation), validated relationships using a correlation matrix, and analyzed weekday vs weekend ridership trends with grouping + nearest-mean day classification.

Key Takeaways

  • Learned how to transform raw real-world CSV data into structured numeric features suitable for modeling and analysis.
  • Used regression performance (R²) to justify a sensor placement decision, and determines which 3-bridge selectiob predicts total traffic with highest accuracy.
  • Confirmed that weather meaningfully influences ridership while weekday commuter patterns dominate overall traffic trends.
