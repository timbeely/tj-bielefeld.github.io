Project Overview

Developed a MATLAB-based cruise control algorithm using noisy sensor data and system modeling techniques to evaluate whether changes in tire characteristics could compromise the stability and safety of an existing cruise control system.

Key Visuals

Technical Stack

  • MATLAB
  • Parameter estimation from noisy sensor data
  • Simulation-based validation

Engineering Approach

  • Modeled cruise control velocity tests using noisy sensor inputs.
  • Estimated system parameters through data analysis and curve fitting.
  • Validated algorithm performance using simulated test cases.

Key Takeaways

  • Learned to extract meaningful parameters from noisy real-world data.
  • Strengthened MATLAB modeling, data analysis, and plot validation skills.
  • Gained experience presenting technical results in a client-ready format.
